# Snowflake ELT Project

## Description
This project implements an ELT (Extract, Load, Transform) pipeline using Snowflake as the cloud data warehouse.

## Features
- Data extraction from various sources
- Loading data into Snowflake
- Data transformation using Snowflake SQL
- Automated pipeline orchestration

## Technologies
- **Snowflake** - Cloud Data Warehouse
- **Python** - Scripting and automation
- **SQL** - Data transformations

## Getting Started

### Prerequisites
- Snowflake account
- Python 3.8+
- Required Python packages (see requirements.txt)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/Pavel404-dev/snowflake-elt-project.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Configure your Snowflake credentials
